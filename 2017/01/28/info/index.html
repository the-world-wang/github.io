commit message

type(scope):subject

type

  • feat:feature新功能
  • fix:bug fix
  • docs:documentation
  • refactor:重构
  • test:测试

scope

我觉得可以普通的项目，可以视为修改的模块

subject

  • 以动词开头
  • 第一个字母小写

change log

change log是一个由人工编辑，以时间为倒叙的列表。记录所有版本的重大改变。

  • 日期：“YYYY-MM-DD”
  • Added
  • Changed
  • Deprecated
  • Removed
  • Fixed
  • Security
