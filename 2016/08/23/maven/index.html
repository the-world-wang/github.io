Posted on 2016-08-23 |

运行main方法

Maven的exec插件允许你运行Java项目中的main方法。

  1. 编译，exec执行时是不会为你编译代码

  2. 运行

概念

groupId是指项目名：公司地址反写+项目名
artifactId指模块名：项目名+模块名
version指版本号
以上3点便是一个坐标

生命周期

常用命令

命令 作用
mvn clena 清除target
mvn install 将jar包安装在本地仓库
mvn archetype:generate 创建maven项目，-DgroupId指定参数
mvn compile 编译主程序代码，不会编译test代码。第一次运行会下载依赖
mvn test-compile 编译测试代码。主程序在classes下面，test在test-classes下面
