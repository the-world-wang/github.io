关于main函数

if __name__=='__main__':
    print('main')

当以单个文件运行.py文件时，__name__便是__main__，而当以模块导入的时候，这个属性便是这个模块的名字。

python虚拟环境

使用pip或者easy_install安装的第三方模块通常安装在site packages目录下。
pyvenv是比较新的工具，使用步骤如下：

  1. pyvenv env_A
  2. source env_A/bin/activate
  3. deactivate

其实原理也很简单，就是改变path而已。
