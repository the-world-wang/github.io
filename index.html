关于main函数

if __name__=='__main__':
    print('main')

当以单个文件运行.py文件时，__name__便是__main__，而当以模块导入的时候，这个属性便是这个模块的名字。

python虚拟环境

使用pip或者easy_install安装的第三方模块通常安装在site packages目录下。
pyvenv是比较新的工具，使用步骤如下：

  1. pyvenv env_A
  2. source env_A/bin/activate
  3. deactivate

其实原理也很简单，就是改变path而已。

内存数据库

jdbc:h2:mem:test;DB_CLOSE_DELAY=-1

  • test: namespace
  • DELAY:默认情况，最后一次连接close，数据库就关闭，加上delay，会知道jvm结束后才关闭。

前言

get started

常用匹配器

core

  1. anything：总是匹配
  2. describedAs:
  3. is

逻辑

  • allOf：所有的匹配器都匹配才匹配
  • anyOf：任意一个匹配器匹配即可
  • both：
  • either：
  • not：如果匹配器不满足

对象

  • equalTo: 对象是否equal
  • instanceOf,isCompatibleType：比较类型
  • notNull,nullValue是否为null

集合

  • hasItem, hasItems：测试一个集合包含一个元素
